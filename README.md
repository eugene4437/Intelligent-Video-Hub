Intelligent Video Hub

Опис проєкту
Інформаційна система для інтелектуальної обробки та семантичного структурування відеоконтенту. Проєкт використовує методи глибокого навчання (YOLOv8, OpenCV) для автоматичного аналізу сцен та розпізнавання об'єктів у відео.

Технології
- Мова: Python 3.11+
- Бібліотеки: OpenCV (обробка кадрів), Ultralytics YOLOv8 (AI детекція), Streamlit (веб-інтерфейс).

Структура репозиторію
- `main.py` — вхідна точка програми (запуск веб-інтерфейсу).
- `src/` — вихідний код з алгоритмами аналізу та обробки.
- `models/` — місце для ваг нейронних моделей.
- `data/` — папка з тестовими відеоматеріалами.
