# Intelligent Video Hub

## Опис проєкту
Система інтелектуальної обробки та семантичного структурування відеоконтенту на основі методів глибокого навчання (YOLOv8, OpenCV).

## Технології
- Python 3.11
- Streamlit (інтерфейс)
- Ultralytics YOLO (нейромережа)

## Структура
- `main.py` — головний файл програми.
- `src/` — вихідний код логіки аналізу.
- `models/` — папка з нейронними мережами.
