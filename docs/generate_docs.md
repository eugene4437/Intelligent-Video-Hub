# Інструкція з генерації документації проєкту

Для автоматичного створення технічного довідника використовується бібліотека **pdoc**.

## 1. Встановлення
`pip install pdoc`

## 2. Генерація HTML
Виконайте команду у корені проєкту:
`pdoc -o ./docs/api main.py`

## 3. Результат
Документація буде збережена у папку `docs/api/`. Відкрийте `index.html` у браузері.
